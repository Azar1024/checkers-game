# Checkers Game

Кроссплатформенная реализация классических русских шашек на **C#** с использованием современного UI-фреймворка **Avalonia**.

Проект создавался в основном в учебных целях в рамках курса ЭСКТ в **НГТУ (Новосибирск)**.

## Основные возможности

- Игровое поле 8×8
- Полная поддержка правил русских шашек:
  - обязательный бой
  - множественные взятия за один ход
  - дамки ходят и бьют на любое количество клеток по диагонали
- Подсветка всех возможных ходов при выборе шашки
- Визуальное выделение последнего хода
- Простая, современная и приятная графика
- Полная кроссплатформенность (Windows / macOS / Linux)

## Технологический стек

- Язык: **C#** (.NET 8)
- UI-фреймворк: **Avalonia** 11.x
- Архитектура: **MVVM** (CommunityToolkit.Mvvm)
- Управление состоянием игры: собственная модель без внешних игровых движков

## Быстрый старт

```bash
# Клонируем проект
git clone https://github.com/Azar1024/checkers-game.git

# Переходим в директорию
cd checkers-game

# Запускаем проект
dotnet run
```
